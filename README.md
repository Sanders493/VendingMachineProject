## General info
This project is a Windows form app that simulates a vending machine. 
It has 2 operating modes:
- Customer Mode
- Maintenance Mode
It comes preloaded with products and can be reloaded.

## Technologies
Project is created with:
* C# 11.0
* .NET 7.0
